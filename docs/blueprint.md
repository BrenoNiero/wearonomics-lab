# **App Name**: Wearonomics Lab

## Core Features:

- Home: Display an introduction to the lab.
- Upload: Provide a page for uploading datasets (currently non-functional).
- Ledger: Provide a page to view the ledger (currently non-functional).
- Research: Provide a page for research documentation (currently non-functional).
- Footer: Display a footer with a copyright notice.

## Style Guidelines:

- Primary color: Desaturated blue (#4285F4) to evoke a sense of trust, calm, and scientific inquiry.
- Background color: White (#FFFFFF) for a clean, minimalist aesthetic.
- Accent color: Pale cyan (#96d6f4) for highlights and subtle interaction cues.
- Font: 'Inter', a grotesque-style sans-serif, for body text and headings to maintain a clean and modern aesthetic.
- Centered content column with a maximum width of 820px to ensure readability.
- Top navigation bar for easy access to Home, Upload, Ledger, and Research pages.
- Use generous whitespace to create a sense of calm and clarity.
- Implement thin, soft dividers to separate content sections.
