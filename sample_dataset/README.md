## Sample Dataset

Wearonomics operates on time-series data derived from wearable devices.

Datasets typically include timestamp, location, speed, and related movement indicators.

The system expects continuous sequences that reflect real-world locomotion. Gaps, inconsistencies, and irregular patterns are evaluated during validation.

This folder provides example datasets to demonstrate how raw input is transformed into validated segments and ledger outputs.

The purpose is not scale, but clarity. Small datasets allow full visibility of each step in the pipeline.
