## System Architecture

Wearonomics is structured as a layered analytical system.

The application layer manages interaction, including data upload, reporting, and ledger visualisation.

The validation layer processes raw wearable data into structured movement segments using explicit rules.

The economic layer converts validated movement into ledger-based outputs through defined attribution models.

The system integrates frontend components, analytical logic, and backend storage into a coherent pipeline.

Each layer operates independently but remains connected through clear data flow.

The architecture prioritises transparency over automation. Every transformation is traceable from input to final output.
